Usage Pattern

Keep the prep flow simple.

This first version is optimized for fast routing, but it already hints at how the site can evolve into a fuller study system.

Recommended Flow
  • Open the topic guide first when you need to refresh the pattern, signals, and decision rules.
  • Jump to the problem list right after to drill the pattern while the heuristics are still fresh.
  • Add new topics by copying the existing card structure so the hub remains predictable.
Easy Next Upgrades
  • Progress badges for each topic, such as not started, revising, and strong.
  • Difficulty or company tags inside each problem page.
  • Links to notes, solution templates, or common mistakes for each section.
