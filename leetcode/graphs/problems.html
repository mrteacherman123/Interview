01 / DFS
Number of Islands

Count connected components of '1's in a char grid. Classic flood fill — each DFS call consumes one island.

DFS BFS Union-Find
Key Insight
DFS from each unvisited '1' cell, mark visited by setting to '0' (mutate). Each DFS call = one island. No visited array needed.
void dfs(char[][] g, int u, int v) {
    g[u][v] = '0'; // mark visited first
    int[] dx={1,-1,0,0}, dy={0,0,1,-1};
    for (int i=0; i<4; i++) {
        int x=u+dx[i], y=v+dy[i];
        if (x<0||y<0||x>=g.length||
            y>=g[0].length) continue;
        if (g[x][y]=='1') dfs(g,x,y);
    }
}
Common Mistake
Setting g[u][v]='0' AFTER the loop causes infinite recursion. Mark visited as the very first line.
Time O(M×N) Space O(M×N)
DFSBFSFlood FillMatrix
02 / BFS + HashMap
Clone Graph

Deep copy an undirected graph where each node has val and a neighbors list. Return the clone of the given node.

BFS + HashMap DFS
Key Insight
HashMap<Node,Node> maps original→clone. BFS traverses; for each neighbor not yet in map, create clone + enqueue. Always add clone link even if already exists.
Map<Node,Node> map = new HashMap<>();
Queue<Node> q = new LinkedList<>();
map.put(node, new Node(node.val));
q.add(node);
while (!q.isEmpty()) {
    Node ns = q.poll();
    for (Node n : ns.neighbors) {
        if (!map.containsKey(n)) {
            map.put(n, new Node(n.val)); q.add(n); }
        map.get(ns).neighbors.add(map.get(n));
    }
}
return map.get(node);
Common Mistake
The neighbors.add() line must be OUTSIDE the if(!containsKey) block — you build neighbor lists even for already-cloned nodes.
Time O(V+E) Space O(V)
BFSDFSHashMapGraph Clone
03 / Topological Sort
Course Schedule

n courses, prerequisite pairs. Return true if you can finish all — i.e., the directed graph has no cycle.

Kahn's BFS DFS Cycle
Key Insight
Kahn's topological sort: if all N nodes are processed → no cycle → return true. If any remain, they're part of a cycle.
int[] indegree = new int[n];
List<List<Integer>> adj = buildAdj(n,prerequisites);
for (int[] p:prerequisites) indegree[p[0]]++;
Queue<Integer> q = new LinkedList<>();
for (int i=0;i<n;i++) if(indegree[i]==0) q.offer(i);
int done=0;
while(!q.isEmpty()){
    done++; int c=q.poll();
    for(int nb:adj.get(c))
        if(--indegree[nb]==0) q.add(nb);
}
return done == n;
Common Mistake
Edge direction: pre[1]→pre[0] (need pre[1] before taking pre[0]). Build adjacency list for O(V+E), not O(VE).
Time O(V+E) Space O(V)
Topological SortDFSCycle Detection
04 / Topological Sort
Course Schedule II

Same setup — return the ordering to take all courses, or empty array if impossible (cycle exists).

Kahn's BFS DFS Post-order
Key Insight
Every time you poll from queue, that course is next in valid order. Build result[] as you poll. Return empty if result.length != n.
int[] result = new int[n]; int idx=0;
// build adj + indegree (same as #03)
Queue<Integer> q = new LinkedList<>();
for(int i=0;i<n;i++) if(indegree[i]==0) q.offer(i);
while(!q.isEmpty()) {
    int c = q.poll();
    result[idx++] = c; // ← record order
    for(int nb:adj.get(c))
        if(--indegree[nb]==0) q.add(nb);
}
return idx==n ? result : new int[0];
Common Mistake
Build adjacency list for O(E) neighbor lookup. Iterating all prerequisites for each node is O(VE).

05 / Hard Topo Sort
Alien Dictionary

Words sorted in alien order. Derive the character ordering from adjacent word pairs via topological sort on chars.

Kahn's BFS
Key Insight
Compare adjacent word pairs, first differing chars give edge c1→c2. Check prefix case: if word1 is longer than word2 but starts with it → return "".
for(int i=0; i<words.length-1; i++) {
    String w1=words[i], w2=words[i+1];
    if(w1.length()>w2.length()
        && w1.startsWith(w2)) return "";
    for(int j=0; j<Math.min(w1.length(),w2.length()); j++){
        char c1=w1.charAt(j), c2=w2.charAt(j);
        if(c1!=c2){
            adj.get(c1).add(c2);
            indegree.merge(c2,1,Integer::sum);
            break; // only first diff!
        }
    }
}
Common Mistake
break after first differing char is mandatory. Forgetting it creates wrong edges. Also: output length must equal total unique chars.
Time O(C) Space O(1)
Topological SortBFSHard
06 / Reverse DFS
Pacific Atlantic Water Flow

Find cells that can flow water to BOTH oceans. Water flows to neighbor if height ≤ current.

Reverse DFS
Key Insight
Reverse flow direction: DFS from Pacific borders + separately from Atlantic borders. Answer = intersection. Climb uphill (neighbor >= current).
void dfs(int[][] g, int u, int v, boolean[][] ocean){
    ocean[u][v] = true;
    for(int i=0;i<4;i++){
        int x=u+dx[i], y=v+dy[i];
        if(x<0||y<0||x>=m||y>=n||ocean[x][y]) continue;
        // reverse: neighbor must be >= current
        if(g[x][y]>=g[u][v]) dfs(g,x,y,ocean);
    }
}
Common Mistake
Condition is g[x][y] >= g[u][v] (reverse flow, climbing uphill). The ocean[x][y] check serves as the visited guard.
Time O(M×N) Space O(M×N)
DFSMatrixReverse Thinking
07 / DFS Border
Surrounded Regions

Flip all 'O' to 'X' except those connected to the border. Classic "mark protected then flip" pattern.

DFS from Borders
Key Insight
DFS from every border 'O', mark as 'P' (protected). Then: P→O, everything else→X. Two-phase approach.
// Phase 1: protect border-connected O's
for(int i=0; i<m; i++) for(int j=0; j<n; j++)
    if((i==0||j==0||i==m-1||j==n-1)
        && board[i][j]=='O') dfs(board,i,j);
// Phase 2: finalize
for(int i=0; i<m; i++) for(int j=0; j<n; j++)
    board[i][j] = board[i][j]=='P' ? 'O' : 'X';
Common Mistake
Starting DFS from every 'O' instead of only BORDER 'O's. Only border-connected O's are safe.
Time O(M×N) Space O(M×N)
DFSMatrixBoundary
08 / Union-Find
Number of Provinces

N×N adjacency matrix. M[i][j]=1 means i and j are friends. Count friend circles (connected components).

Union-Find DFS
Key Insight
Connected components from adjacency matrix. UF: start count=N, union(i,j) when M[i][j]=1, decrement count. Final count = number of provinces.
int count = n;
UF uf = new UF(n);
for(int i=0;i<n;i++)
    for(int j=i+1;j<n;j++)
        if(M[i][j]==1 && !uf.connected(i,j)) {
            uf.union(i,j); count--; }
return count;
Common Mistake
DFS version: iterate over n (not m×n) since input is adjacency matrix, each row = one person. Only traverse upper triangle j=i+1 to avoid double-counting.
Time O(N²) Space O(N)
Union-FindDFSConnected Components
09 / Union-Find
Number of Connected Components

n nodes, edge list. Count connected components in undirected graph given as edge pairs.

Union-Find DFS
Key Insight
Union-Find is most elegant: start with n components, union(u,v) for each edge, decrement count only when two different components merge.
int count = n;
UF uf = new UF(n);
for(int[] e: edges)
    if(!uf.connected(e[0],e[1])) {
        uf.union(e[0],e[1]); count--; }
return count;
// find() with path halving for ~O(1) per op
Common Mistake
DFS version needs adjacency list — build map from edges first. Watch for NPE on isolated nodes with no edges.
Time O(V+E) Space O(V)

Transform beginWord to endWord one char at a time, each intermediate word must be in wordList. Return shortest path length.

BFS Bidirectional BFS
Key Insight
Bidirectional BFS: two sets from begin and end. Always expand the SMALLER set. When word appears in other set → path found. Time: b^(d/2) vs b^d.
Set<String> bS=new HashSet<>(), eS=new HashSet<>();
bS.add(beginWord); eS.add(endWord); int len=1;
while(!bS.isEmpty())&&!eS.isEmpty()) {
    if(bS.size()>eS.size()) {var t=bS; bS=eS; eS=t;}
    Set<String> tmp=new HashSet<>();
    for(String w: bS) {
        char[] c=w.toCharArray();
        for(int i=0;i<c.length;i++) for(char x='a';x<='z';x++){
            char old=c[i]; c[i]=x; String nw=new String(c);
            if(eS.contains(nw)) return len+1;
            if(dict.contains(nw)) { tmp.add(nw); dict.remove(nw); }
            c[i]=old; }
    }
    bS=tmp; len++;
}
Common Mistake
Mutate char array in-place and RESTORE after each trial. Remove from dict instead of separate visited set to avoid revisiting.

11 / BFS Coloring
Is Graph Bipartite?

Undirected graph as adjacency list. Return true if it can be 2-colored (no two adjacent nodes same color).

BFS Coloring Union-Find
Key Insight
BFS 2-coloring: assign color 0 to start. Each uncolored neighbor gets the opposite color. If neighbor already has same color → not bipartite.
int[] color = new int[n]; Arrays.fill(color,-1);
for(int i=0; i<n; i++) {
    if(color[i]!=-1) continue; // already colored
    color[i]=0; Queue<Integer> q=new LinkedList<>(); q.add(i);
    while(!q.isEmpty()) {
        int u=q.poll();
        for(int v: graph[u]) {
            if(color[v]==color[u]) return false;
            if(color[v]==-1){ color[v]=1-color[u]; q.add(v); }
        }
    }
}
return true;
Common Mistake
Graph may be DISCONNECTED — must start BFS from every unvisited (color==-1) node, not just node 0.
Time O(V+E) Space O(V)

12 / DFS + Memo
Longest Increasing Path in Matrix

m×n matrix. Move 4 directions only to strictly larger values. Find longest increasing path length.

DFS + Memo
Key Insight
Strictly increasing path → no cycles → DFS safe without visited array. cache[i][j] = longest path from (i,j). Memoize to avoid recomputation.
int[][] cache = new int[m][n];
int dfs(int[][] mat, int u, int v) {
    if(cache[u][v]!=0) return cache[u][v];
    for(int i=0;i<4;i++){
        int x=u+dx[i], y=v+dy[i];
        if(x<0||y<0||x>=m||y>=n) continue;
        if(mat[x][y]>mat[u][v])
            cache[u][v]=Math.max(cache[u][v],dfs(mat,x,y));
    }
    return ++cache[u][v]; // +1 for current cell
}
Common Mistake
Don't use a visited array — it would prevent finding optimal paths through shared cells. Strictly increasing guarantees no cycles.
Time O(M×N) Space O(M×N)
DFSMemoizationMatrixDP
13 / Multi-source BFS
01 Matrix

Binary matrix. For each cell return distance to nearest 0. Multi-source BFS is the key insight here.

Multi-source BFS
Key Insight
Seed ALL 0 cells at distance 0. BFS outward — each 1 cell's distance set when first reached (guaranteed minimum). One pass over entire matrix.
Queue<int[]> q = new LinkedList<>();
for(int i=0;i<m;i++) for(int j=0;j<n;j++)
    if(mat[i][j]==0) q.offer(new int[]{i,j});
    else mat[i][j]=Integer.MAX_VALUE;
while(!q.isEmpty()) {
    int[] p=q.poll();
    for(int d=0;d<4;d++){
        int x=p[0]+dx[d], y=p[1]+dy[d];
        if(!valid(x,y)) continue;
        if(mat[x][y]>mat[p[0]][p[1]]+1)
            { mat[x][y]=mat[p[0]][p[1]]+1; q.add(new int[]{x,y}); }
    }
}
Common Mistake
Setting 1-cells to MAX_VALUE serves as "unvisited" marker — the > check updates only unvisited or suboptimal cells.
Time O(M×N) Space O(M×N)
Multi-source BFSMatrixShortest Path
14 / Multi-source BFS
Walls and Gates

Grid with -1 (wall), 0 (gate), INF (empty). Fill each empty room with distance to nearest gate.

Multi-source BFS
Key Insight
Same as 01 Matrix. Seed all gate cells (value 0) at once. BFS outward — rooms get minimum distance automatically. Never touch walls (-1).
int INF = Integer.MAX_VALUE;
Queue<int[]> q = new LinkedList<>();
for(int i=0;i<m;i++) for(int j=0;j<n;j++)
    if(rooms[i][j]==0) q.add(new int[]{i,j});
while(!q.isEmpty()) {
    int[] p=q.poll();
    for(int d=0;d<4;d++){
        int x=p[0]+dx[d], y=p[1]+dy[d];
        if(!valid(x,y)||rooms[x][y]!=INF) continue;
        rooms[x][y]=rooms[p[0]][p[1]]+1; q.add(new int[]{x,y});
    }
}
Common Mistake
Check rooms[x][y] != INF to skip walls (-1) AND already-updated rooms. Don't need a separate visited array.
Time O(M×N) Space O(M×N)

15 / DFS
Minesweeper

'M'=mine, 'E'=empty, click cell. If mine→'X', else count adj mines; if 0 reveal neighbors recursively.

DFS 8-dir
Key Insight
Count adjacent mines FIRST. If count > 0 → set digit, stop. If count = 0 → set 'B', DFS all 8 'E' neighbors. Order matters!
void dfs(int u, int v) {
    int cnt=0;
    for(int i=0;i<8;i++){ int x=u+dx[i],y=v+dy[i];
        if(valid(x,y) && board[x][y]=='M') cnt++; }
    if(cnt>0) { board[u][v]=(char)(cnt+'0'); return; }
    board[u][v]='B'; // only recurse if cnt==0
    for(int i=0;i<8;i++){ int x=u+dx[i],y=v+dy[i];
        if(valid(x,y) && board[x][y]=='E') dfs(x,y); }
}
Common Mistake
Must precompute mine count BEFORE deciding whether to recurse. The board[x][y]=='E' check prevents revisiting revealed cells.
Time O(M×N) Space O(M×N)
DFS8-directionsMatrix
16 / BFS
Shortest Path in Binary Matrix

n×n grid of 0s and 1s. Find shortest 8-directional clear path from (0,0) to (n-1,n-1).

BFS 8-dir
Key Insight
Standard BFS with 8 directions. Store distance in grid in-place (set to distance value). Edge case: start or end cell is 1 → return -1.
if(grid[0][0]!=0||grid[n-1][n-1]!=0) return -1;
if(n==1) return 1; // single cell
Queue<int[]> q=new LinkedList<>(); q.add(new int[]{0,0}); grid[0][0]=1;
while(!q.isEmpty()) {
    int[] p=q.poll(); int d=grid[p[0]][p[1]];
    for(int i=0;i<8;i++){
        int x=p[0]+dx[i],y=p[1]+dy[i];
        if(x<0||y<0||x>=n||y>=n||grid[x][y]!=0) continue;
        if(x==n-1&&y==n-1) return d+1;
        grid[x][y]=d+1; q.add(new int[]{x,y});
    }
}
return -1;
Common Mistake
8 directions (not 4)! Also check BOTH start and end cells for being 1. Special case: n=1 returns 1 before BFS.
Time O(N²) Space O(N²)
BFS8-directionsShortest Path
17 / DFS
Max Area of Island

Binary matrix. Find maximum area (count of 1s) among all islands. Elegant recursive counting.

DFS
Key Insight
Elegant recursive counting: return dfs(up)+dfs(down)+dfs(left)+dfs(right)+1. The +1 is for the current cell. Base case: OOB or not 1 → return 0.
int dfs(int u, int v) {
    if(u<0||v<0||u>=m||v>=n||mat[u][v]!=1) return 0;
    mat[u][v]=-1; // mark visited (-1 != 1 or 0)
    return dfs(u,v+1) + dfs(u+1,v)
         + dfs(u-1,v) + dfs(u,v-1) + 1;
}
// main: max = Math.max(max, dfs(i,j))
// for each cell where mat[i][j]==1
Common Mistake
Mark as -1 (not 0) to distinguish from original water cells if needed. The elegant one-liner pattern includes the +1 for current cell.
Time O(M×N) Space O(M×N)
DFSFlood FillMatrix
18 / Math
Island Perimeter

Single island in grid. Calculate perimeter — each land cell contributes 4 edges minus shared edges with neighbors.

Math / Counting
Key Insight
For each land cell start with 4. Subtract 1 for each adjacent land neighbor. Only check UP and LEFT to avoid double-counting: each shared edge removes 2.
int result = 0;
for(int r=0;r<rows;r++)
    for(int c=0;c<cols;c++)
        if(grid[r][c]==1) {
            result+=4;
            // only check UP and LEFT (avoid double-count)
            if(r>0 && grid[r-1][c]==1) result-=2;
            if(c>0 && grid[r][c-1]==1) result-=2;
        }
return result;
Common Mistake
Checking all 4 neighbors and subtracting 1 each is also correct (total ends up same), but only UP+LEFT with -=2 is cleaner and avoids double-counting.
Time O(M×N) Space O(1)
MatrixMathEasy
19 / DFS Rolling
The Maze

Ball rolls until hitting a wall. Can it stop at destination? DFS on stopping positions, not individual cells.

DFS BFS (shortest)
Key Insight
Roll in a direction until hitting a wall — that's a stopping position. Only track stopping positions as visited. Inner while-loop handles the rolling.
void dfs(int u, int v) {
    visited[u][v]=true;
    if(u==dest[0]&&v==dest[1]) { found=true; return; }
    for(int i=0;i<4;i++) {
        int x=u, y=v;
        // roll until wall
        while(x+dx[i]>=0&&y+dy[i]>=0&&
               x+dx[i]<m&&y+dy[i]<n&&maze[x+dx[i]][y+dy[i]]==0)
            { x+=dx[i]; y+=dy[i]; }
        if(!visited[x][y]) dfs(x,y);
    }
}
Common Mistake
The ball doesn't stop at every cell — only at walls. Mark only STOPPING positions as visited. The inner while rolls through intermediate cells.
Time O(MN·max(M,N)) Space O(MN)
DFSBFSMatrixRolling
20 / Dijkstra
Minimum Cost Path (Weighted Grid)

Grid with heights. Cost to move = abs(height diff). Find minimum cost from (0,0) to (m-1,n-1).

Dijkstra Priority Queue
Key Insight
Non-uniform edge weights → BFS fails. Dijkstra with PQ ordered by cumulative cost. When same cell is dequeued twice, skip if already visited.
PriorityQueue<int[]> pq=new PriorityQueue<>((a,b)->a[2]-b[2]);
int[][] dist=new int[m][n]; for(int[] r:dist) Arrays.fill(r,Integer.MAX_VALUE);
dist[0][0]=0; pq.add(new int[]{0,0,0});
while(!pq.isEmpty()) {
    int[] c=pq.poll();
    if(vis[c[0]][c[1]]) continue; vis[c[0]][c[1]]=true;
    for(int d=0;d<4;d++) {
        int x=c[0]+dx[d],y=c[1]+dy[d];
        if(!valid(x,y)||vis[x][y]) continue;
        int nd=c[2]+Math.abs(h[x][y]-h[c[0]][c[1]]);
        if(nd<dist[x][y]) { dist[x][y]=nd; pq.add(new int[]{x,y,nd}); }
    }
}
return dist[m-1][n-1];
Common Mistake
if(vis[c[0]][c[1]]) continue is critical — same cell may be enqueued multiple times with different costs; only process it once.
Time O(MN·logMN) Space O(MN)
DijkstraPriority QueueWeighted Graph
21 / DFS + Backtrack
Robot Room Cleaner

Robot API: move(), turnLeft(), turnRight(), clean(). No map given. Clean every reachable cell.

DFS + Backtrack
Key Insight
Track relative (row,col,direction). (d+i)%4 converts direction index to absolute. After DFS: goBack = turn180, move, turn180. Always turnRight after each attempt.
void backtrack(int row, int col, int d) {
    robot.clean(); visited.add(new Pair(row,col));
    for(int i=0;i<4;i++) {
        int nd=(d+i)%4;
        int nr=row+dr[nd], nc=col+dc[nd];
        if(!visited.contains(new Pair(nr,nc)) && robot.move()) {
            backtrack(nr,nc,nd); goBack(); }
        robot.turnRight(); // always rotate to next dir
    }
}
void goBack() { robot.turnRight(); robot.turnRight();
    robot.move(); robot.turnRight(); robot.turnRight(); }
Common Mistake
turnRight() is called EVERY iteration (whether move succeeded or not) to cycle through all 4 directions. goBack restores position+direction.
Time O(N-M) Space O(N-M)

22 / Euler Path
Reconstruct Itinerary

Airline tickets [from,to]. Start from JFK. Use all tickets exactly once. Return lexicographically smallest itinerary.

Hierholzer's DFS Post-order
Key Insight
Hierholzer's algorithm: DFS with PriorityQueue (sorted neighbors for lex order). Add to result in POST-ORDER (addFirst). Dead-ends are handled naturally.
Map<String,PriorityQueue<String>> graph = new HashMap<>();
for(String[] t:tickets)
    graph.computeIfAbsent(t[0],x->new PriorityQueue<>()).add(t[1]);
LinkedList<String> path = new LinkedList<>();
void dfs(String from) {
    PriorityQueue<String> q = graph.get(from);
    while(q!=null && !q.isEmpty()) dfs(q.poll());
    path.addFirst(from); // POST-ORDER
}
dfs("JFK"); return path;
Common Mistake
addFirst (not add) — result is built in reverse post-order. Without this, dead-end branches appear too early in the result.
Time O(E log E) Space O(E)
DFSEuler PathHard
23 / In-place
Game of Life

Conway's Game of Life. Update all cells simultaneously. Do it in O(1) extra space.

In-place encoding
Key Insight
Encode transitions: live→dead = -1, dead→live = 2. Count live neighbors by checking Math.abs(board[x][y])==1. Two-pass approach.
for(int i=0;i<m;i++) for(int j=0;j<n;j++) {
    int live=countLive(board,i,j); // |v|==1 or v==-1
    if((board[i][j]==1||board[i][j]==-1) // was alive
        &&(live<2||live>3)) board[i][j]=-1;
    if(board[i][j]==0 && live==3) board[i][j]=2;
}
for(int[] r:board) for(int c=0;c<n;c++)
    r[c] = r[c]>0 ? 1 : 0;
Common Mistake
When counting neighbors, include -1 values (they were alive). Math.abs(v)==1 counts both 1 and -1 as live. Use 8 directions.
Time O(M×N) Space O(1)

25 / BFS
Remove Invalid Parentheses

Remove minimum invalid parentheses. Return all valid strings. BFS guarantees minimum removals.

BFS Level-order
Key Insight
BFS level = number of chars removed. First level with any valid string is the minimum. Set found=true, collect all at that level, stop going deeper.
Queue<String> q=new LinkedList<>(); Set<String> vis=new HashSet<>();
q.add(s); vis.add(s); boolean found=false;
while(!q.isEmpty()) {
    String cur=q.poll();
    if(isValid(cur)) { found=true; res.add(cur); }
    if(found) continue; // don't expand further
    for(int i=0;i<cur.length();i++) {
        if(cur.charAt(i)!='('&&cur.charAt(i)!=')') continue;
        String t=cur.substring(0,i)+cur.substring(i+1);
        if(vis.add(t)) q.add(t);
    }
}
Common Mistake
found flag lets current-level nodes report valid strings but stops NEXT-level generation. Without it you'd generate extra removals.

Key Insight

    for(int nb:adj.get(node))
        if(--indeg[nb]==0) q.add(nb);
}

All strings are anagrams. Two strings are "similar" if they differ in exactly 0 or 2 positions. Count groups.

Key Insight
For each pair, count differing positions. If diff is 0 or 2 → similar → union them. Since all are anagrams, diff=1 is impossible.
UF uf=new UF(n);
for(int i=0;i<n;i++) for(int j=i+1;j<n;j++)
    if(isSimilar(strs[i],strs[j])) uf.union(i,j);

boolean isSimilar(String a, String b) {
    int diff=0;
    for(int i=0;i<a.length();i++)
        if(a.charAt(i)!=b.charAt(i)&&++diff>2) return false;
    return true; // diff==0 or diff==2
}

diff==1 is NOT similar (anagrams can't differ in exactly 1 position since character counts must match). Early exit when diff > 2.

Intersection of Two Island Grids

Two binary grids a and b. Count islands that exist in exactly the same coordinates in BOTH grids.

DFS dual-grid
Key Insight
DFS when a[i][j]==1 OR b[i][j]==1. Track if entire island matches (all cells have same value in both). Count only fully-matching islands.
boolean dfs(int u, int v) {
    visited[u][v]=true;
    // both must be 1 for a match
    boolean res = a[u][v]==1 && b[u][v]==1;
    for(int d=0;d<4;d++) {
        int x=u+dx[d],y=v+dy[d];
        if(!valid(x,y)||visited[x][y]) continue;
        if(a[x][y]==1||b[x][y]==1)
            // res on RIGHT to avoid short-circuit!
            res = dfs(x,y) && res;
    }
    return res;
}
Common Mistake
res = dfs(x,y) && res NOT res && dfs(x,y) — the latter short-circuits and skips DFS on remaining cells, leaving them unvisited!
Time O(M×N) Space O(M×N)
